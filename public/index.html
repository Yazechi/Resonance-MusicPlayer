Setup Required

A required tool is missing. Resonance needs both yt-dlp and mpv to play music.

Windows (winget):
winget install yt-dlp.yt-dlp
winget install mpv.net

macOS (brew):
brew install yt-dlp mpv

Linux:
sudo apt install mpv && pip install yt-dlp

After installing, restart the server and refresh this page.
